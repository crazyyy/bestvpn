== Description ==
Extends the functionality of [WP REST API]. Allows you to request only certain fields.

Use &fields=ID,title to fetch only the ID and title. Works with posts, comments, taxonomy, media.

[WP REST API]: https://wordpress.org/plugins/json-rest-api/
